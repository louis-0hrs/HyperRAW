# HyperRAW

High-performance RAW video conversion tool with initial focus on BRAW workflows.

HyperRAW is a professional, GPU-accelerated converter designed for speed and
batch-oriented pipelines.

Current format support:
- BRAW (production-ready)

Planned:
- Additional RAW formats (e.g. RED)

Distribution and licensing are handled externally.
